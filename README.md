# Poseidon project 

![](logo.png)


This is the Poseidon restapi project


## versions
- python : 2.7
- django : 1.10.5
- django rest_framework : 3.5.3


## build & run
```bash
git clone [git repo addr]

cd  poseidon-rest

docker build -t poseidon-rest:v0.1 .

docker run -d -p 8000:8000 poseidon-rest
```
